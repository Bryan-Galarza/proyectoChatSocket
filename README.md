Proyecto de chat en tiempo real utilizando nodejs y socket io, dispone de ingreso con username real o aleatorio, notificaciones, vista del chat y vista de usuarios conectado. Se utilizó una bd creada en turso con dos tablas una para chat general y otra para el chat privado. El proyecto esta desplegado en railway http://proyectochatsocket-production.up.railway.app/

pero se puede ejecutar en local con npm run dev en http://localhost:3000/

Las variables de entorno para la base de datos turso se configuran en el archivo .env que está enlazado correctamente con el archivo index.js.
